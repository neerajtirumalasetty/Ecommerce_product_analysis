# E-commerce Pricing & Demand Analysis

## Problem
Analyzed how pricing, product ratings, categories, and brands influence product demand in an e-commerce setting.

## Approach
- Cleaned and transformed raw product data
- Created demand proxy using rating count and purchase signals
- Performed exploratory data analysis to identify patterns
- Analyzed demand across price segments, categories, and brands

## Key Insights
- Low-priced products show significantly higher demand
- Medium-priced products maintain consistent performance
- Ratings influence demand but are not a strong differentiator
- Essential apparel categories dominate demand
- A few value-driven brands capture majority of engagement

## Tools Used
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)
- Jupyter Notebook

## Outcome
Identified key drivers of demand and provided insights for pricing and product strategy optimization.
